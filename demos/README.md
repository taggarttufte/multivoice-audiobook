# Demo

Three short audio clips generated by this pipeline from
[*Pride and Prejudice*](https://www.gutenberg.org/ebooks/1342) by Jane Austen
(public domain, via Project Gutenberg).

| File | Length | Source | What it shows |
|------|-------:|--------|---------------|
| [opening.mp3](./opening.mp3)   | 35 s | Chapter I    | The famous opening line, then Mr. & Mrs. Bennet's first exchange |
| [lucas.mp3](./lucas.mp3)       | 45 s | Chapter VI   | Multi-character scene — Elizabeth, Darcy, Bingley, Caroline |
| [proposal.mp3](./proposal.mp3) | 60 s | Chapter XXXIV | Darcy's first proposal — heated dialogue between two characters |

Cast for the demo (auto-detected by `cast_builder.py`, with major-role
overrides):

- **Elizabeth** → `eve`
- **Darcy** → `rex`
- **Jane** → `ara`
- **Bingley** → `leo`
- **Narrator** → `sal`
- minor characters → gender-matched pool (`eve`/`ara` for women, `rex`/`leo`
  for men), assigned by hash so each minor character is consistent within
  the book

## Regenerating the demo

The full demo costs roughly **$0.07** in Grok TTS calls. If you've already
rendered once, the per-segment cache means re-runs cost essentially nothing.

```bash
# 1. Download the EPUB into this folder (gitignored)
curl -L -o demos/pg1342.epub https://www.gutenberg.org/ebooks/1342.epub3.images

# 2. Run the demo builder
python demos/build_demo.py
```

`build_demo.py` does everything end-to-end: extract book text, detect cast,
attribute dialogue, render each chapter with multi-voice TTS, trim with
ffmpeg. See the script for what's pluggable (sample list, chapter ranges,
trim lengths).
